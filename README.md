A MATLAB code that can generate discrete fault networks, as described in the Im (2023) article. Please note that it should not be used for commercial purposes. Reference:
>> Im, K., & Avouac, J.-P. (2023). Cascading foreshocks, aftershocks and earthquake swarms in a discrete fault network. Geophysical Journal International, 235(1), 831–852. https://doi.org/10.1093/gji/ggad278
